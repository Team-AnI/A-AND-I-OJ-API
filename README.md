# aandi_oj_api

A&I OJ(Online Judge) API 클라이언트 패키지입니다.

## 개요
- 패키지명: `aandi_oj_api`
- 런타임: Dart ^3.10.4
- 주요 의존성: `dio`, `aandi_api_protocol`, `aandi_auth`, `freezed_annotation`

## 설치

```yaml
dependencies:
  aandi_oj_api:
    git:
      url: git@github.com:Team-AnI/A-AND-I-OJ-API.git
      ref: v0.1.0
```

## 사용법

```dart
import 'package:aandi_oj_api/aandi_oj_api.dart';
```
